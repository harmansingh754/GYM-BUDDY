# 🏋️ GymBuddy – Smart Workout & Gym Partner Finder

GymBuddy is a modern fitness web application developed using **HTML, CSS, and Vanilla JavaScript**.
The project is specially designed for college students to manage workouts, track fitness progress, save gym schedules, and find compatible workout partners.

---

# 🚀 Features

## 🏠 Dashboard

* Attractive hero section
* Daily motivational quotes using Fetch API
* Push / Pull / Leg workout logic
* Fitness tips section
* Responsive navigation bar

## 💪 Workout Planner

* Push Day workouts
* Pull Day workouts
* Leg Day workouts
* Exercise cards with images
* Filter functionality using JavaScript

## 📅 Timetable Manager

* Add free gym timings
* Save slots using localStorage
* Dynamic rendering of schedules
* Form validation

## 🤝 Gym Buddy Finder

* Match gym partners with similar schedules
* Match percentage system
* Dynamic profile cards
* “Workout Together” interaction button

## 📈 Profile & Progress

* BMI Calculator
* Workout streak tracker
* User profile section
* Progress tracking

## 🌙 Dark / Light Mode

* Theme toggle using JavaScript
* Saved theme preference using localStorage

## 🔐 Authentication

* Signup page
* Login page
* Frontend authentication using localStorage
